# Image Downloader


## Install, run and testing

### Requirements:

- rvm or rbenv
- ruby 3.1.0
- bundler version 2.2.32

### Pre-run steps:
#### Pull the project
```
$ git clone git@github.com:Zandelok/image_downloader.git
```
#### Open the project
```
$ cd image_downloader/
```
#### Install all dependencies
```
$ bundle
```

### Usage

#### Grant permissions:

```
$ chmod +x bin/main
```

#### Running the tests:

```
$ rspec
```

#### Running the script:

```
$ bin/main public/input_data.txt public/images/
```
